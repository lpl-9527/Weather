天气预报(Weather)
===========
天气预报是一款基于Android端开源的天气预报软件，具备查看全国的省市县、查询任意城市天气、自由切换城市、手动更新天气、后台自动更新天气等功能。天气预报中的天气数据由和风天气提供，背景图片由必应提供，代码遵循Apache v2 License开源协议。本软件主要是本人学习《第一行代码》后运用教程知识所完成的app。
